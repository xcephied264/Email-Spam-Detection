# Email-Spam-Detection
Developed a machine learning model to classify email messages as spam or legitimate
Preprocessed textual data using TF-IDF vectorization
Implemented and compared models including Naive Bayes and Support Vector Machine (SVM)
Evaluated performance using accuracy, precision, recall, and F1-score
Achieved high classification accuracy with improved spam detection performance

SVM achieved 98% accuracy with strong precision and recall balance
